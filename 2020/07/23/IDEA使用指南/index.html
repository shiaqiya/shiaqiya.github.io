前言：本篇博客介绍 idea开发工具的一些基础常规配置、java项目的创建、快捷键、小技巧。

个人从eclipse转到idea 将使用的过程记录下来以备查看。

参考的博文会统一放在文章底部。

1、项目的创建

1.1、启动页面介绍

1.2、idea中的 项目、模块与eclipse中的工作空间，项目的异同

工作空间只是存放文件的地方，idea中新建项目存储在工作空间文件夹里，这里面很干净的，项目名一一对应，没有别的文件。项目里面新建模板，一个项目可对应多个模板，每个模板独立

✦ IntelliJ IDEA 没有类似 Eclipse 的工作空间的概念（Workspaces），最大单元就是
Project。

➊ Eclipse中的Workspace:每一个WorkSpace中的配置都是独立的，一个WorkSpace中可以有多个Project项目,Eclipse打开的是WorkSpace,运行时可以选择具体运行哪个项目

➋ IntelliJ IDEA中是以单个项目为单位的，每一个项目都是独立的

➋ 虽然也可以用IDEA打开多个项目的上级目录，在页面中也会显示类似Eclipse中的目录结构，但是此时IDEA会将这个主目录文件当成是一个项目，而主目录下的Project当成是项目的子文件

1.3、IntelliJ IDEA创建第一个Java项目

  1. Create New Project 或者 file–> new –> project
  2. 选择指定目录下的 JDK 作为 Project SDK，如果要创建 Web工程，则需要勾选 Web Application。如果不需要创建 Web工程的话，则不需要勾选。这里先不勾选，只是创建简单的Java工程。
  3. 点击Next
  4. 输入项目名，选择项目存放位置，然后点击Finish，注意Project name和Project location路径要一致（ 建议直接在Project location后面加上创建的项目名路径，让Project name自动同步 ）
  5. 至此第一个项目已经创建完毕，其他操作就类似Eclipse中的类似了。

1.4、IntelliJ IDEA创建Module

  1. 在Eclipse中我们有Workspace（工作空间）和 Project（工程）的概念，在IDEA中只有Project（工程）和Module（模块）的概念。这里的对应关系为：

​ ◌ 从 Eclipse 转过来的人总是下意识地要在同一个窗口管理n个项目，这在IntelliJ IDEA是无法做到的。IntelliJ IDEA提供的解决方案是打开多个项目实例，即打开多个项目窗口。即：一个 Project 打开一个 Window 窗口。

​ ◌ 在IntelliJ IDEA中Project是最顶级的级别，次级别是Module。一个 Project可以有多个Module。目前主流的大型项目都是分布式部署的，结构都是类似这种多Module结构。
